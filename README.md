?Repositorio criado para aplicar, testar e aperfeiçoar habilidades de desenvolvimento em C.

A idéia deste projeto é aperfeiçoar os conhecimentos na linguagem C e aplicar em um jogo simples para console de comando, com gráficos simples, podendo evoluir para gráficos melhores.

Os objetivos principais do projeto são:
-Usar os conhecimentos básicos e evoluir;
-Usar acesso a arquivo para salvamento de dados do jogo;
-Inserir comentários nos trechos de código sempre que necessário, com o intuito de documentar o código;
-Realizar controle de versão.

****
This repository was created to apply, test and improve development skills in C.

The main project idea is to improve the knowledge in C and apply them in a simple bash console game, with simple grafics, able to go ahead with better grafics.

The main goals are:
-Use the basic knowledge and evolve;
-Use file access to save game data;
-Insert code comments as needed, in order to generate documentation;
-Do the version control.
